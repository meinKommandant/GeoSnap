# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in GeoSnap, please report it responsibly:

1. **Do NOT** open a public GitHub issue for security vulnerabilities.
2. Send an email to the maintainers with details of the vulnerability.
3. Include steps to reproduce the issue if possible.

We will respond within 48 hours and work with you to understand and address the issue.

## Security Best Practices

When using GeoSnap:

- Only process photos from trusted sources
- Be cautious when opening Excel files from unknown origins
- Keep your Python environment and dependencies updated
